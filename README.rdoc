= Find a Sponsor
== Instructions for running
  
Install gems:

  bundle install

Configure database: fill 'username' and 'password' in 'config/database.yml'

Create database:

  bundle exec rake db:setup

Configure environments: create file config/environment_variables.yml with following content
  development:
  GMAIL_USERNAME: name@gmail.com
  GMAIL_PASSWORD: password
  
Seed:
  rake db:seed

Run sidekiq:
  sidekiq -q mailer -c 5

Run application:
  rails s
