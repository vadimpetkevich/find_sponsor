= News Portal
== Configuration

Dependencies:

  sudo apt-get install imagemagick postgresql
  
Install gems:

  bundle install

Configure database:

  bundle exec rake db:setup
  
Populate test database

  bundle exec rake db:seed RAILS_ENV='test'
  
Run test
  
  bundle exec rspec spec
